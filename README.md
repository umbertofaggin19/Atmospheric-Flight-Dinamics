# Atmospheric-Flight-Dinamics
script9.m = 3 DoF simulator for 3d flight of ligth airplane
  Manovre implementate:
  Virata corretta nel piano orizzontale
  Loop verticale 
  Salita/discesa nel piano verticale
  Spirale ad asse verticale e ad asse orizzontale (Barrel Roll)

  Requisiti
  Matlab (R2020a o successive)

  Per avviare la simulazione:
  Digitare "script9.m" nella command window di Matlab
